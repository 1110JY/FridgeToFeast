# Fridge to Feast

An ingredient-first recipe discovery platform. Instead of searching for a dish, you select what you already have and the app surfaces the best matches.

## Features

- Ingredient selection organised by category (proteins, vegetables, spices, etc.)
- Recipe matching ranked by ingredient overlap percentage
- Dietary preference filtering
- Nutritional metadata display
- Async API handling with loading states and error feedback

## Tech Stack

- React
- Node.js + Express
- External Recipe API (Spoonacular)
